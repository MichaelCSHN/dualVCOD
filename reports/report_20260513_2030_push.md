## 5. How to Reconstruct Full Experiment State

```bash
# 1. Clone repo
git clone https://github.com/MichaelCSHN/dualVCOD.git
cd dualVCOD

# 2. Generate resized dataset (required for all training)
python tools/generate_resized_dataset.py

# 3. Reproduce any trial (example: E-45 baseline)
python tools/autoresearch/run_trial_minimal.py \
  --trial_id expl_45_baseline_hard_8ep \
  --config configs/autoresearch/expl_45_baseline_hard_8ep.json

# 4. Run unified reeval
python tools/reeval_backbone_probes.py
python tools/reeval_teacher_route.py

# 5. Generate error audit CSVs
python tools/run_error_audit.py

# 6. Generate flatfish_2 annotated frames
python tools/e43_model_vs_mask.py
```
